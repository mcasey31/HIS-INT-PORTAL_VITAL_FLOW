# HU HOME-AD-01 - Accesos Directos

## Trazabilidad
- Epic: EPICA HOME
- Feature: FEATURE_HOME
- Tipo: User Story
- Estado: New
- URL: pendiente

## User Story
Como usuario del HIS
quiero ver una pantalla Home con accesos directos
para ingresar rapidamente a los modulos principales.

## Descripcion
La pantalla Home debe mostrar una seccion "Accesos directos" con tarjetas de acceso a:
- Personas
- Gestion de Agendas
- Turnos
- Escritorio Clinico

Cada tarjeta debe incluir titulo, descripcion y accion "INGRESAR".

## Criterios de Aceptacion
- Se visualizan las 4 tarjetas en Home.
- Personas y Gestion de Agendas navegan al modulo correspondiente.
- Turnos y Escritorio Clinico quedan visibles para habilitarse en iteraciones siguientes.
