# FRONT

Esta carpeta contiene exclusivamente el desarrollo de interfaz de usuario.

## Incluye
- Pantallas y componentes UI.
- Navegacion y estado de cliente.
- Validaciones de formulario de nivel presentacion.
- Integracion con APIs del BACK.

## No incluye
- Reglas de negocio core.
- Acceso directo a base de datos.
- SQL o migraciones.
