## Azure Descripcion
Como: Gestor de agendas. Quiero: Eliminar una regla de cupo. Para: Eliminar las reglas y/o condiciones creados para el bloque de programación. 

 Descripción y comportamiento: Como gestor de agenda, es necesario poder eliminar las reglas y/o condiciones de los cupos dentro de un bloque de programación. Para ello, se requiere que en la lista de gestión de cupos se incluya una opción que permita eliminar estas reglas. 
 
 
 
 Al eliminar esta acción, se borrará el registro de la regla
establecida para ese bloque de programación y su condición ya no será tomada en
cuenta en la gestión de turnos en el futuro, antes de proceder, se mostrará un
mensaje de confirmación.
