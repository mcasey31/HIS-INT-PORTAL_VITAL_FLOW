## Azure Descripcion
Como: Gestor de turnos Quiero: Tener configurado en base de datos las indicaciones previas por prácticas Para: Consultarlas al momento de asignar un turno al paciente 

 
 Descripción y comportamiento: 
 
 Desde el sistema asistencial de ODI, se requiere la configuración y adecuación de todas las practicas a nivel de base de datos para que tengan identificadas las indicaciones previas que debe tener en cuenta el paciente en cualquier ámbito asistencial. Ej. Ecografía abdominal Indicaciones Previas: - Retención de Liquido de 2 Hs. 
- Debe concurrir con ayuno de 12 Hs.

## Azure Criterios de Aceptacion
- Se debe crear una estructura a nivel de Base de Datos donde se almacenes las indicaciones previas y se relacionen a las practicas.
