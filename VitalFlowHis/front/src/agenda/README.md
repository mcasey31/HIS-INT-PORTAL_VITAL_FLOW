# Modulo Front Agenda

Este modulo consume exclusivamente APIs del backend.

## Contratos usados
- GET /api/v1/agendas
- POST /api/v1/agendas
- POST /api/v1/agendas/{agendaId}/bloques
- POST /api/v1/agendas/{agendaId}/bloqueos
- GET /api/v1/agendas/{agendaId}/disponibilidad

## Nota
- Componente base: AgendaPage.tsx
- Cliente API: agendaApi.ts
