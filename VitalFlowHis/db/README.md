# DB

Esta carpeta contiene el modelo de datos y su evolucion.

## Incluye
- Scripts de migracion.
- Definicion de tablas, indices y constraints.
- Seeds para ambientes de prueba.
- Scripts de versionado y rollback.

## No incluye
- Logica de UI.
- Reglas de negocio de aplicacion (deben vivir en BACK).
