# Proceso Funcional E2E

Esta carpeta concentra la documentacion funcional y tecnica del avance implementado en este chat para el flujo E2E de Estructura, Convenios y Facturacion.

## Contenido

1. `01_estado_funcional_chat.md`
   - Resumen ejecutable de lo construido en este chat.
   - Cambios de UX, backend y reglas de auditoria.
   - Alcance confirmado y pendientes.

2. `02_componentes_prestaciones_catalogos.md`
   - Mapa funcional de componentes.
   - Relacion Catalogo ODI y catalogos de financiador (OSDE, SWISS, SWISS-SMG20, VITALSALUD).
   - Ciclo de vida de prestaciones y homologaciones.

3. `03_flujo_operativo_e2e.md`
   - Flujo punta a punta recomendado (alta de maestros hasta facturacion).
   - Checkpoints de validacion y control.
   - Riesgos operativos y criterios de salida.

## Objetivo de uso

- Evitar perdida de contexto entre iteraciones.
- Servir como base de homologacion funcional y UAT.
- Alinear a equipo funcional, tecnico y QA sobre el mismo recorrido E2E.
