## Azure Descripcion
Como usuario efector medico 

Quiero poder agregar opcionalmente a cada practica una observacion 

Para actualizar las evoluciones y enviar las ordenes medicas al paciente. 

 
 Junto a cada practica figura el menú de opciones para "Agregar obsevación" Una vez que se agrega la observación se visualiza junto a la páctica un icono de "información" y se habilita el menú de edición o eliminar.

## Azure Criterios de Aceptacion
Solo se podran agregar observaciones a las practicas ya asignadas a una orden (fecha). El límite de caracteres por observación es de 140. El botón de guardar onservación se habilita cuando se haya ingresado texto. El menú solo muestra editar y eliminar observación cuando exista una previamente, sino solo la opción "Agregar observación" Luego de agregar, modificar o eliminar debe mostrarse un mensaje en la esquina superior derecha indicando el cambio realizado.
