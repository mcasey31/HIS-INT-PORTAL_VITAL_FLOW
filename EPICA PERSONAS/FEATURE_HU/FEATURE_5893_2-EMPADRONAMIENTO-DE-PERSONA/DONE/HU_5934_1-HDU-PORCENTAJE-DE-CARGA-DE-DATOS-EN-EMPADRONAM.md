## Azure Descripcion
Como: Usuario de sistema ODI 

 Quiero: Ver porcentaje de avance en carga de datos 

 Para: Completar datos de carga 

 

 Descripción y comportamiento: 

 Desde la pantalla de empadronamiento, en la parte superior se visualizará un marco que nos indicará el porcentaje de avance en la carga de datos de la persona a empadronar (pantalla) según el siguiente criterio: 

 Set de datos mínimos 50% 

 
Datos de contacto 30% 

 
Dirección 10% 

 
Persona de contacto 10% 

 
 Link a pantallas: 

 (pantalla) Marco de carga de datos

## Azure Criterios de Aceptacion
Criterios de aceptación: 

 -Al avanzar en la carga de datos se irá incrementando el porcentaje de valor en el marco.
